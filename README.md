## 🚀 Sobre o Curso
O curso abrange desde a lógica de programação básica até tópicos avançados de Java, incluindo:
* Lógica de Programação e Estruturas de Controle.
* Programação Orientada a Objetos (Classes, Atributos, Métodos, Sobrecarga, Encapsulamento, Herança e Polimorfismo).
* Tratamento de Exceções.
* Interfaces, Generics, Set e Map.
* Programação Funcional e Expressões Lambda.
* Acesso a Banco de Dados com JDBC e JPA/Hibernate.
* Desenvolvimento Web com Spring Boot.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java 17+
* **IDE:** IntelliJ IDEA
* **Versionamento:** Git e GitHub

## 📂 Estrutura do Repositório
O projeto está organizado por seções conforme a estrutura do curso:
* `Sec4_EST_Seq`: Estrutura Sequencial
* `Sec8_POO`: Introdução à Programação Orientada a Objetos
* `Sec10_Arrays_list`: Comportamento de memória, arrays e listas
* `Sec13_Heranca_Polimorfismo`: Pilares da POO
